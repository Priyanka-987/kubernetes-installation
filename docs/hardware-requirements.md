# Hardware Requirements

## Minimum (lab/testing)
- 2 vCPU
- 4 GB RAM
- 20 GB disk

## Recommended (small production)
- Control plane: 4 vCPU, 8-16 GB RAM, 50+ GB disk
- Worker nodes: 4+ vCPU, 8+ GB RAM
- Network: reliable LAN, unique IPs for each node

## Notes
- For high-availability you must have multiple control-plane nodes (not covered by these scripts).
